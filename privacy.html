PitchBox Privacy Policy v1.0

Last updated: April 12, 2026 | Effective date: April 12, 2026

Summary: PitchBox reads publicly visible HTML source code of websites you visit to detect technology signatures. We do NOT collect personal data of website visitors, do NOT scrape private information, and do NOT sell any data. Your settings and saved prospects are stored locally on your device.

1. Who We Are

PitchBox is a Chrome browser extension developed and operated by DivergeiX ("we", "us", "our"). PitchBox helps sales development representatives (SDRs) and outbound teams analyze publicly available website technology information and generate personalized outreach content.

For any privacy-related queries, contact us at: privacy@divergeix.com

2. What Data We Collect

2.1 Website Technology Data (Processed Locally)

When you click "Scan This Page", PitchBox reads the publicly visible HTML source code of the active browser tab to identify:

  • Technology signatures (e.g., script tags referencing jQuery, React, WordPress)
  • Meta tags (page title, description, Open Graph tags)
  • Publicly visible company metadata (company name from meta tags, domain name)
  • Presence of common web tools (analytics, marketing, chat widgets)

2.1.1 Deep Scan Feature

When you explicitly click "Deep Scan", PitchBox fetches up to 10 additional publicly accessible subpages on the same website (e.g., /about, /pricing, /team) to gather more complete information. This feature:

  • Only runs when you explicitly click the "Deep Scan" button - never automatically
  • Checks and respects the website's robots.txt file before scanning
  • Skips pages marked with noindex meta tags
  • Limits to 10 subpages maximum per deep scan
  • Rate-limits requests to 1 page every 1.5 seconds to avoid server load
  • Only scans pages on the same domain - never follows external links
  • All data is processed locally in your browser

Important: This is functionally identical to a user pressing Ctrl+U (View Page Source) in their browser. We only read publicly served HTML. We do NOT access cookies, localStorage, session data, passwords, form inputs, or any private data on any website.

2.2 Data Stored Locally on Your Device

The following data is stored in your browser's local storage (chrome.storage.local) and never leaves your device unless you explicitly choose to sync:

  • Your PitchBox settings (name, company name, role, preferences)
  • Your Claude API key (stored locally, used only for direct API calls from your browser)
  • Saved prospect records (company profiles you choose to save)
  • Scan history (last 20 scanned domains)
  • Theme preference (dark/light mode)
  • Generated outreach drafts (stored in session, cleared when browser closes)

2.3 Account Data (Stored on Our Servers)

If you create a PitchBox account for paid features, we collect and store:

  • Email address - For account identification and login
  • Password - Stored as a bcrypt hash (we never store plaintext passwords)
  • Usage counters - Daily scan and draft counts for quota enforcement
  • Payment records - Transaction IDs from Razorpay (we do NOT store credit card numbers)
  • Account plan status - Free or Pro

This data is stored in Microsoft Azure Cosmos DB (Central India region) with encryption at rest.

2.4 Data We Do NOT Collect

  • Personal data of website visitors or website owners
  • Browsing history beyond the pages you explicitly scan
  • Cookies or session data from any website
  • Form inputs or passwords from any website
  • Credit card or banking information (Razorpay handles payment processing)
  • Location data, IP addresses, or device fingerprints
  • Data from tabs you do not explicitly scan
  • Background browsing activity

3. How We Use Your Data

  • Technology detection: To identify web technologies from publicly visible HTML and display results to you
  • Outreach generation: To generate personalized sales messages using the Anthropic Claude API (your API key, sent directly from your browser)
  • Account management: To authenticate your login and enforce usage quotas
  • Payment processing: To process subscription payments through Razorpay

4. Third-Party Services

4.1 Anthropic Claude API

When you generate outreach drafts, your browser makes a direct API call to Anthropic's Claude API using your personal API key. We do not proxy, store, or have access to these API calls. Anthropic's privacy policy applies to this data processing: anthropic.com/privacy

4.2 Razorpay

Payment processing is handled by Razorpay. We do not store credit card numbers or banking details. Razorpay is PCI DSS compliant. Their privacy policy: razorpay.com/privacy

4.3 Microsoft Azure

Account data and usage records are stored in Azure Cosmos DB (Central India). Azure is SOC 2, ISO 27001, and GDPR compliant. Their privacy policy: privacy.microsoft.com

5. Data Retention

  • Local data: Stored until you uninstall the extension or clear browser data
  • Session data (drafts): Cleared when you close the browser
  • Account data: Retained while your account is active. Deleted within 30 days of account deletion request
  • Usage logs: Daily usage records are retained for 90 days for billing purposes

6. Your Rights (GDPR & Global Privacy)

You have the right to:

  • Access: Request a copy of all data we hold about you
  • Rectification: Correct any inaccurate data
  • Deletion: Request complete deletion of your account and all associated data
  • Portability: Export your saved prospects and settings
  • Restriction: Restrict processing of your data
  • Objection: Object to processing of your data
  • Withdraw consent: Uninstall the extension at any time to stop all data processing

To exercise any of these rights, email privacy@divergeix.com. We will respond within 30 days.

7. Data Security

  • Passwords are hashed using bcrypt (never stored in plaintext)
  • Authentication uses JWT tokens with 30-day expiry
  • All API communication uses HTTPS/TLS encryption
  • Azure Cosmos DB provides encryption at rest
  • API keys are stored locally in your browser only
  • We do not have access to your Claude API key

8. Chrome Extension Permissions

PitchBox requests the following Chrome permissions:

  • activeTab: To read the HTML of the page you're currently viewing when you click Scan
  • sidePanel: To display the PitchBox analysis panel alongside the webpage
  • storage: To save your settings and prospects locally
  • scripting: To inject the page scanner when you click Scan
  • tabs: To identify which tab to scan
  • contextMenus: To add "Analyze with PitchBox" to the right-click menu

We request host_permissions for all URLs so the scanner can work on any website you visit. The scanner only runs when you explicitly click "Scan This Page" - never in the background.

9. Children's Privacy

PitchBox is a professional sales tool and is not intended for use by individuals under 18 years of age. We do not knowingly collect data from minors.

10. Changes to This Policy
